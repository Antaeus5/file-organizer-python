#  File Organizer in Python

A simple Python script that automatically organizes files into folders based on their type (images, documents, videos, etc.).

##  Features
- Automatically classifies files by extension
- Creates folders if they don't exist
- Prevents file overwriting
- Easy to use

##  Technologies
- Python
- os
- shutil

##  How to Use

1. Clone the repository:
   git clone https://github.com/Antaeus5/file-organizer-python.git

2. Navigate to the folder:
   cd file-organizer-python

3. Run the script:
   python organizador.py

4. Enter the folder path you want to organize

##  Example

Input:
C:\Users\User\Downloads

Output:
- Images/
- Documents/
- Videos/
- Others/
